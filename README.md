**Peer-reviewed publications from the rENM Framework research program**

## Overview

This repository collects the published and in-preparation papers underlying
the rENM Framework — a modular suite of R packages for reconstructing and
analyzing long-term ecological niche dynamics using historical species
occurrence records and environmental data.
